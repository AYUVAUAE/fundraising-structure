01 — SAFEノートとは

Simple Agreement for Future Equity（将来株式取得契約）

投資家資金を提供
→資金
SAFEノート株式取得の権利
（株式そのものではない）
⇢将来転換
株式所定タイミングで転換

基本情報

  • 米国シリコンバレー発祥、日本では J-KISS として VC 投資の標準
  • 「今資金を預かり、将来の所定タイミングで株式に転換」する契約
  • 投資時点では株式ではなく「株式取得の権利」を保有
  • 採用理由：企業価値の確定を遅らせ迅速な手続きが可能

早期投資家のメリット

  • バリュエーションキャップ：将来の評価額に上限を設定 → 早期ほど有利
  • ディスカウント：将来ラウンドの価格から割引適用 → 早期ほど多く株式取得
  • 先行リスクへのリターンを契約に組み込み、後発投資家との公平性を確保

重要 金商法49人ルールは完全に適用

  • SAFEノートは金商法上「みなし有価証券」に該当
  • 株式ではなく権利でも、実質的な有価証券として規制対象
  • 1オファリングあたり日本人投資家 49名 の上限がそのまま適用

「SAFEなら49人ルールを回避できる」という誤解は誤り。
→ 50名以上は公募登録が必要（5,000万〜2億円のコスト）

02 — AYUVAコーポレートストラクチャー

ホールディング配下に投資・建設・運営の3レイヤー、その下に複数SPV

親子関係 資金提供 建設業務／委託料 運営業務 委託料（還流）

レイヤー間のフロー

  • 投資会社 → SPV：資金提供
  • プロジェクト管理 → SPV：建設業務 ／ SPV → プロジェクト管理：建設委託料
  • アセット管理 → SPV：運営業務 ／ SPV → アセット管理：運営委託料

03 — 88M AED Year 1 シナリオ

SPV収益 88M AED に対する各レイヤーへの分配

①② 資金の流れ（投入） ③④⑤ 還流・配当 ⑥ 管理料（同オーナー間） 運営費
  1. 投資家 → 投資会社（SAFEノート、初期資金提供）
  2. 投資会社 → SPV（資本配分）／ SPV内：運営費 50% ＋ アセット管理料 5% 控除
  3. SPV → 投資会社（純利益 39.6M AED 還流）
  4. 投資会社 → 投資家（配当 19.8M 優先支払 / ティア別）
  5. 投資会社 → ホールディング（残額 19.8M AED）
  6. アセット管理会社 → ホールディング（管理料 4.4M AED、同一オーナー）

05 — 49人ルール（金商法）の課題

日本人投資家からの資金調達における最大の構造的制約

制約 制約条件

  • 日本では1オファリングあたり投資家 49人 までが上限
  • SAFEノートにも完全に適用される（回避不可）
  • 公募登録はコスト（5,000万〜2億円）と時間（6〜12ヶ月）から見送り

検討中の合法的3ルート

1

海外投資家への分散

対象地域：
  • UAE、GCC諸国
  • シンガポール、香港
  • ヨーロッパ
→ 49人ルール対象外 ／ 制限なし
2

フェーズ型オファリング

3ラウンド構成：
  • プレ・プレセール
  • プレセール
  • メインセール
→ 各6ヶ月の間隔／ 各49人＋実質的差別化
3

日本の登録ブローカー起用

第二種金融商品取引業者ライセンス
  • コスト：3〜7%
  • ＋ セットアップ費用
→ 49人超の規模拡大可

ルート2に必須：6ヶ月沈黙期間ルール

✕ 禁止事項 — 日本人投資家への勧誘一切

  • メール・DM
  • ピッチ・プレゼン
  • 口頭での投資言及
  • ロードショー
  • 次回ラウンド予告
  • ウェイトリスト構築
  • SNS投資関連発信
  • 「将来投資できますよ」発言
  • 第三者経由の紹介
  • 既存投資家への Phase 2 勧誘

勧誘と取られる行為すべて沈黙期間中は不可
違反 → 構造全体崩壊リスク

✓ 継続可能な活動 — 事業運営は通常通り

  • ブランドマーケティング
  • 顧客（ゲスト・会員）募集
  • 非日本人投資家への募集（UAE、GCC、欧州、他アジア）
  • 既存投資家への運営報告
  • 建設・運営業務
  • 採用・チームビルディング
  • パートナーシップ構築
  • PR・メディア露出（事業内容の紹介に限定）

「投資の話をしないなら、基本的に何でもOK」
事業運営に支障なし

ルート3の詳細：ブローカー・ディーラールート（調査中）

  • ライセンス：第二種金融商品取引業者
  • 想定コスト：調達資本の 3〜7% ＋ セットアップ 500万〜3,000万円 ＋ 年次継続費
  • 紹介：弁護士から適格業者の紹介予定
  • トレードオフ：コスト vs 49人超の規模拡大可能性

次のステップ：業者調査中・候補リストと見積取得

推奨アプローチ：3ルートの組み合わせ

  • ルート1（海外投資家）＋ ルート2（フェーズ型）を基本戦略
  • ルート3（ブローカー起用）は規模拡大時のオプションとして検討
  • 全ルートともに弁護士の書面意見および法令遵守体制が前提

補足：ルート2は沈黙期間ルールが必須要件 ／ ルート1・ルート3は沈黙期間の制約を直接受けない（独立した運用が可能）
